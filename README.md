# SAFARI EN – WhatsApp Bot

## Features
- Beautiful bold menu
- AI (ChatGPT) & Image Generation
- Music & Video downloads
- Group management (kick, add, promote, demote, warn, ban)
- Anti-link, anti-badword, anti-emoji
- Welcome/Goodbye image cards
- Economy & Leveling system
- Daily rewards, games
- Premium user system
- Multi-language (English / Shona)

## Deploy on Render
1. Fork this repo
2. Create a new Web Service on Render
3. Connect your repo, set environment variables
4. Deploy!
